# Excel_Dashboard
Interactive Excel dashboard analyzing Vrinda Store’s 2022 sales performance, customer demographics, regional trends, and channel-wise revenue using Pivot Tables and data visualization techniques.

### Vrinda Store Annual Report 2022 – Excel Dashboard

This project presents an interactive Excel dashboard analyzing the 2022 annual sales performance of Vrinda Store. The dashboard provides actionable business insights through data cleaning, transformation, and visualization using advanced Excel features.

##### Project Objectives:
1. Analyze monthly sales vs orders trend
2. Compare sales contribution by gender
3. Evaluate order status distribution
4. Identify top 5 performing states
5. Analyze age group vs gender purchasing behavior
6. Examine channel-wise order contribution

##### Tools & Techniques Used :
1. Microsoft Excel
2. Pivot Tables
3. Pivot Charts
4. Slicers (Month, Channel, Category filters)
5. Data Cleaning & Formatting
6. Conditional Formatting
7. KPI Analysis

##### Dashboard Features:
1. Orders vs Sales Trend Analysis
2. Sales: Men vs Women Comparison
3. Order Status Breakdown
4. Top 5 States by Revenue
5. Age Group vs Gender Analysis
6. Channel-wise Sales Distribution

##### Key Business Insights:

--> Women contribute the majority of sales (64%).

--> 92% of orders were successfully delivered.

--> Maharashtra is the top-performing state.

--> Amazon generates the highest channel contribution.

--> Adult women segment drives maximum revenue.
